To describe a job, please fill in a set of tasks separated by “;”. Each task must be defined with the machine it must be executed on and a duration as follow (machine, duration).

Example:

   Job 1 is composed of three task respectively executable on machine 0, 1 and 2 (blue) with a duration of 3, 2 and 2 (red).

Please keep in mind the following restriction:
   - Do not add “;” after the last task.
   - Do not forget “;” between each task.
   - Do not add more than 9 tasks to a job.
   - Use a single digit to define your machine and your duration (max 9).
