Max-Min Ant System (MMAS) is a variant of ACO algorithms proposed by Thomas St√ºtzle and Holger H. Hoos. It aims to enhance ACO performances thanks to a stronger exploitation of the best solution known. The algorithm structure is the same as AS. However, MMAS will differs in three points:

  • Only the best ant will perform the pheromone update. Best ant can be defined as the iteration best or the global best (best from the beginning).
  • Pheromone concentration is bounded ([τmin; τmax]) to limit stagnation.
  • The initial value of pheromones is set to τmax, to encourage exploration at the beginning of the algorithm.
